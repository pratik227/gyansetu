જ્ઞાનસેતુ GyanSetu

The first AI tutor that actually works in rural Gujarat — because it doesn't need the internet. Powered by Gemma 4 running fully on-device via Google AI Edge LiteRT.

🌍 5 Indian languages 📵 Fully offline 🧠 Multi-tool agent 📷 Vision + Voice ⚖️ Apache 2.0

What it does

📷

Camera Scan

Point at any object. Gemma 4's multimodal vision identifies it bilingually with phonetic + a one-line story for the kid to learn.

🎙

Voice Q&A

Speak in Gujarati, Hindi, Marathi, Tamil or English. Audio goes straight to Gemma 4 — no separate STT — answer streams back live with offline TTS readback.

🧠

Multi-tool agent

Eight on-device tools. The agent chains query_syllabus → generate_quiz → mark_homework in a single turn. All tool calls grounded in local Room SQLite — zero hallucination on curriculum facts.

👩‍🏫

Teacher Mode

Multi-student roster, per-topic mastery matrix, AI-generated focus recommendations, exportable progress reports — all offline, shareable via Bluetooth / WhatsApp.

🏆

Quiz + Match Game

Adaptive difficulty driven by the kid's mastery scores. Confetti celebration. Streaks, badges, achievements that follow the student across sessions.

♿

Accessibility-first

WCAG-AAA high-contrast palette, dyslexia-friendly font option, three reading levels (Std 1-2 / 3-4 / 5), font-size scaling, audio-first navigation for non-readers.
